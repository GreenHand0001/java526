# 《幼儿园管理系统》 java526

## 项目概述

本项目为基于前后端分离架构的幼儿园管理系统，旨在通过信息化手段提高幼儿园的管理效率，实现家长与学校的良好互动。

## 技术栈

- 架构：B/S、MVC
- 开发环境：IDEA、JDK1.8、Maven
- 数据库：Mysql
- 技术框架：Java、SpringBoot、Mybatis、Vue

## 功能模块

系统包含以下四种用户角色及其功能模块：

### 管理员用户
- 职工信息管理

### 后勤主任用户
- 个人信息管理
- 职工请假审批
- 考勤登记汇总
- 制定食谱信息
- 缺失物品信息管理

### 教师用户
- 个人信息管理
- 幼儿信息管理
- 请假审批提交与审批
- 课程信息安排
- 幼儿活动管理
- 考勤打卡
- 提交缺失物品信息
- 特殊情况批复
- 班级信息管理
- 体检信息管理

### 家长用户
- 个人信息管理
- 提交幼儿信息
- 请假审批提交
- 查看教师信息
- 查看幼儿食谱信息
- 查看幼儿体检信息

## 系统角色
- 管理员用户
- 后勤主任用户
- 教师用户
- 家长用户
